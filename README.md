# Financial Performance Insight Engine (DuPont Analysis)

## Project Overview
This project develops a rule-based financial insight engine using the DuPont framework to analyze and interpret company performance.  
It combines financial decomposition, growth analysis, and automated insight generation to transform raw financial data into actionable business intelligence.

---

## Objective
- Decompose Return on Equity (ROE) into key drivers  
- Identify whether performance is driven by profitability, efficiency, or leverage  
- Generate structured insights for decision-making  
- Compare financial performance across companies and time  

---

## Dataset
- Source: Public financial data (annual reports / financial platforms)  
- Companies: 5 Indian FMCG companies  
  - Hindustan Unilever  
  - ITC  
  - Nestlé India  
  - Britannia  
  - Dabur  
- Time Period: FY20 – FY25  
- Observations: 30 (5 companies × 6 years)

---

## ⚙️ Tools & Technologies
- Python (Pandas, NumPy)  
- Power BI (Visualization & dashboards)  
- Excel (Data preparation)  

---

## Methodology
1. Data Collection & Cleaning  
2. Financial Computation:
   - Profit Margin (PM)  
   - Asset Turnover (AT)  
   - Equity Multiplier (EM)  
   - Return on Equity (ROE)  
3. Growth Analysis (YoY changes)  
4. Classification Framework (Low / Moderate / High)  
5. Rule-Based Insight Generation  
6. Visualization using Power BI  

---

## Key Insights
- Financial performance is driven by different strategies across firms (profitability, efficiency, or leverage)  
- High ROE does not always indicate strong fundamentals; leverage can inflate returns  
- Profitability-driven firms show more stable performance  
- Efficiency and leverage-driven firms show higher variability and risk
